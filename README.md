# How to build application on local
### 1.Clone this repository
```
git clone https://github.com/ItinoseSan/MyPage.git
```
### 2.Check the ruby version on rbenv
```
rbenv versions
```
### 3.Import bundler and install bundle
```
gem install bundler
bundle install
```
### 4.Try on rails local server
```
rails server
```
